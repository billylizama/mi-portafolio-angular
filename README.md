# Mi Portafolio — Billy Lizama

Portafolio personal desarrollado con Angular, desplegado en GitHub Pages.

## Demo en vivo

[Ver portafolio](https://billylizama.github.io/mi-portafolio-angular/)

## Tecnologías

| | Tecnología |
|---|---|
| Framework | Angular 21 |
| Lenguaje | TypeScript |
| Estilos | CSS3 |
| Deploy | GitHub Pages |

## Instalación local

```bash
# Clonar repositorio
git clone https://github.com/billylizama/mi-portafolio-angular.git
cd mi-portafolio-angular

# Instalar dependencias
npm install

# Ejecutar en desarrollo
ng serve
```
Abre `http://localhost:4200` en tu navegador.
