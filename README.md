# Portafolio Personal — Billy Lizama

> Portafolio profesional de Desarrollador Full Stack construido con **Angular 21**, componentes standalone y arquitectura de servicios. Desplegado en GitHub Pages.

## Demo en vivo

**[billylizama.github.io/mi-portafolio-angular](https://billylizama.github.io/mi-portafolio-angular/)**

## Stack

| Área | Tecnología |
|---|---|
| Framework | Angular 21 (Standalone Components) |
| Lenguaje | TypeScript |
| Estilos | CSS3 con Custom Properties |
| Fuentes | Plus Jakarta Sans + Inter |
| Deploy | GitHub Pages |
| Control de versiones | Git Flow |

## Características

- Navbar fija con efecto blur al hacer scroll
- Hero a pantalla completa con animaciones CSS
- Sección "Sobre Mí" con skills organizados por categoría
- Cards de proyectos con tech stack y hover effects
- Sección de contacto con links directos
- Diseño 100% responsivo (mobile-first)

## Instalación local

```bash
git clone https://github.com/billylizama/mi-portafolio-angular.git
cd mi-portafolio-angular
npm install
ng serve
```

Abre `http://localhost:4200` en tu navegador.
